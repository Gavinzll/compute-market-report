# Compute Market Report

这个仓库用于发布每日《算力三指数/六指数日度报告》的 HTML 网页版。

## 页面结构

- `index.html`：报告首页和历史入口
- `latest.html`：最新日报，飞书每天优先推送这个链接
- `reports/`：按日期归档的历史日报
- `data/`：后续可存放历史数据、指数数据和趋势数据
- `assets/`：后续可存放图表、样式和图片资源

## 推荐访问地址

启用 GitHub Pages 后，最新报告地址通常为：

`https://gavinzll.github.io/compute-market-report/latest.html`
